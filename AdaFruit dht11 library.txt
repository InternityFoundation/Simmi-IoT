Before running the python file DHT11READ.py on python script write the following on terminal to download AdaFruitDHT11 library 


>>sudo apt-get install git-core
>>git clone https://github.com/adafruit/Adafruit_Python_DHT.git
>>cd Adafruit_Python_DHT
>>sudo apt-get install build-essential python3-dev  # python3
>>sudo python3 setup.py install  # python3
>>python DHT11READ.py
